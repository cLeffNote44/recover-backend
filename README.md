### Installation (3 steps)

1. **Extract this package** to your desired location

2. **Install dependencies:**
   ```bash
   cd source
   npm install
   ```

3. **Run the app:**
   ```bash
   npm run dev
   ```

4. **Open in browser:**
   Navigate to `http://localhost:5173`

That's it! The app is now running locally.
